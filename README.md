# ShareBoost — เครื่องมือแชร์โพสต์ Facebook ไปกลุ่มอัตโนมัติ

## วิธีติดตั้ง (Windows)

### ขั้นตอนที่ 1: ติดตั้ง Node.js
1. ไปที่ https://nodejs.org/
2. โหลด LTS version (ปุ่มเขียวใหญ่ทางซ้าย)
3. ติดตั้งตามปกติ (กด Next ไปเรื่อยๆ)

### ขั้นตอนที่ 2: ติดตั้ง Google Chrome (ถ้ายังไม่มี)
- โหลดจาก https://www.google.com/chrome/
- ต้องมี Chrome หรือ Edge อย่างใดอย่างหนึ่ง

### ขั้นตอนที่ 3: ติดตั้ง ShareBoost
1. แตก zip ไปที่ไหนก็ได้ (เช่น C:\ShareBoost)
2. เปิด Command Prompt (กด Win+R พิมพ์ cmd)
3. พิมพ์:
```
cd C:\ShareBoost
npm install
npm start
```

### ขั้นตอนที่ 4: ใช้งาน
1. ไปที่หน้า **ตั้งค่า** (⚙️)
2. กด **"เพิ่มบัญชี"** → จะเปิด Chrome ให้ login Facebook
3. Login Facebook ตามปกติ → ระบบจะจำ session ไว้
4. กลับมาหน้า **แชร์** (🚀)
5. เลือกบัญชี → ใส่ URL โพสต์ → เลือกกลุ่ม → กดแชร์!

## ฟีเจอร์
- ✅ แชร์โพสต์ไปหลายกลุ่มอัตโนมัติ
- ✅ AI สร้าง Caption (ต้องใส่ API Key ใน Settings)
- ✅ Spintax {คำ1|คำ2|คำ3} สุ่มเปลี่ยนคำ
- ✅ จัดกลุ่มเป็นหมวดหมู่
- ✅ หน่วงเวลาสุ่มเหมือนคนจริง
- ✅ รายงานสถิติ

## AI Caption (ถ้าต้องการ)
1. ไปที่ ตั้งค่า > AI Caption
2. เลือก Provider: OpenRouter / DeepSeek / OpenAI
3. ใส่ API Key
4. กด "ทดสอบ" ดูว่าใช้ได้
5. กลับหน้าแชร์ → เปิด AI Caption → เลือกสไตล์ → สร้าง!

## หมายเหตุ
- ใช้ Chrome/Edge ที่ติดตั้งในเครื่อง (ไม่ต้องลง browser เพิ่ม)
- ข้อมูลทั้งหมดเก็บในเครื่องคุณเท่านั้น (SQLite)
- ไม่ส่งข้อมูลไปที่ไหน
