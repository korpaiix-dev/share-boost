# ShareBoost — แอดมินเพจ AI ทำงาน 24 ชม.

## Product Positioning

ShareBoost ไม่ใช่แค่ซอฟต์แวร์ — แต่เป็น **"แอดมินเพจ AI"** ที่ทำงานแทนคนจริง 24/7

ราคาจ้างคนดูแลเพจในไทย:

| บริการ | ราคา/เดือน | ข้อจำกัด |
|--------|-----------|---------|
| ฟรีแลนซ์ราคาถูก | 3,000-5,000 บาท | โพสวันละครั้ง, ตอบแชทช้า, ไม่มีวันหยุดก็หยุด |
| เอเจนซี่กลาง | 5,000-10,000 บาท | คอนเท้น 20 ชิ้น, ตอบแชทเวลาทำการ |
| เอเจนซี่ Premium | 10,500-35,000 บาท | ครบวงจร + ยิงแอด + วิดีโอ |

**ShareBoost AI ทำได้มากกว่า ในราคาถูกกว่า:**
- ทำงาน 24 ชม. ไม่มีวันหยุด
- ตอบแชทภายใน 5 วินาที (คนจริงตอบ 5-30 นาที)
- ไม่ลาป่วย ไม่ลาออก ไม่ต้องจ่ายประกันสังคม
- คิดคอนเท้นต์ไม่มีวันหมดไอเดีย

## สิ่งที่ต้องมีทาง Server (Openclaw)

1. **License API** — activate, verify, usage tracking, billing
2. **Facebook Graph API Proxy** — Page management, post scheduling, insights
3. **Messenger Webhook** — รับ/ส่งแชท real-time
4. **Comment Webhook** — รับคอมเม้นใหม่ real-time
5. **AI Service** — LLM API สำหรับ Content/Chat/Comment (DeepSeek/OpenAI)
6. **Video Script Service** — สร้าง script + คำบรรยาย
7. **User Dashboard** — web portal สำหรับดูสถิติ + จัดการ license
8. **Database** — PostgreSQL: users, licenses, pages, chat_logs, content, analytics
9. **Payment Gateway** — Stripe / PromptPay / บัตรเครดิต
